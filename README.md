# East Asian History Comp Review D&D 5E Project

This is the Website made for Q4 East Asian Hisory Comp Review D&D 5E Project.
This project is made by: Benjamin Dai, Amy Chen, Ashley Huang, Jacky Zeng, Teresa Jiang

## What is this?

This is a website that allows you to review the East Asian History Comp Review (Sengoku Era) D&D 5E Project.

## How to develop

1. Install `pnpm`
2. Run `pnpm install`
3. Run `pnpm run dev`
4. Open `http://localhost:5173/` in your browser
